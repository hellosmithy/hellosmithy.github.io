# hellosmithy.com

Personal website and blog, built with [Astro](https://astro.build/) using the [Zaduma](https://github.com/hasparus/zaduma) template.

## Running Locally

```sh
pnpm install
pnpm dev
```

## Building

```sh
pnpm build
```

## Writing Posts

Add MDX files to `posts/`. Include `title` and `date` in frontmatter:

```mdx
---
title: "Post Title"
date: "2026-01-01"
---

Content goes here.
```
